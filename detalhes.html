Sobre a TechStore

A TechStore é um mini e-commerce dinâmico desenvolvido como projeto acadêmico prático. A plataforma consome catálogos estruturados em JSON para renderizar componentes de interface de forma 100% dinâmica através de Vanilla JS, CSS customizado e componentes do framework Bootstrap 5.
